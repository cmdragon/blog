## Passaggio 1: Accedere a AI Magic Resume

Apri il sito web AI Magic Resume:  
**https://magic-resume.cmdragon.cn/app/dashboard/ai**

Seleziona l'opzione **"Fornitore di IA"** nella barra di navigazione sinistra.

![Seleziona fornitore di IA](https://cdn-cn.cmdragon.cn/aBClo3u1L9-mfxakQY_6H.png)

## Passaggio 2: Ottenere la chiave API OpenRouter

### 2.1 Aprire il sito web OpenRouter (usando OpenRouter come esempio)

Nella pagina di configurazione del fornitore di IA, fai clic sul pulsante **"Ottieni chiave API"**.

![Fai clic per ottenere la chiave API](https://cdn-cn.cmdragon.cn/syW9LMwLw0mBa1zn4p7DI.png)

Il sistema aprirà il sito ufficiale OpenRouter in una nuova scheda:

![Sito ufficiale OpenRouter](https://cdn-cn.cmdragon.cn/kJmBIJzNwEna_Dqixs5Gh.png)

### 2.2 Registrarsi/Accedere

- **Hai già un account**: Accedi direttamente
- **Non hai un account**: Fai clic per registrarti

> 💡 **Suggerimento**
> : Se non vuoi usare la tua email abituale, puoi usare il nostro [servizio di email temporanea](https://tools.cmdragon.cn/zh/apps/temp-email) https://tools.cmdragon.cn/zh/apps/temp-email (assicurati che OpenRouter supporti le email temporanee).

![Servizio di email temporanea](https://cdn-cn.cmdragon.cn/Dn0WKPCb4GJRJzJTViZMl.png)
![Registrare un account](https://cdn-cn.cmdragon.cn/zGVLipU7YiUpQ8LinfcsJ.png)

### 2.3 Creare una chiave API

Dopo aver effettuato l'accesso, segui questo percorso:

1. Fai clic su **"Settings"** nella barra di navigazione superiore
2. Seleziona **"API Keys"** dal menu sinistro
3. Fai clic sul pulsante **"Create API Key"**

![Pagina delle impostazioni](https://cdn-cn.cmdragon.cn/lKVZ6Umc3EkhQRowk7O_E.png)
![Gestire le chiavi API](https://cdn-cn.cmdragon.cn/B6lzexbzcIwpT6oltMdNr.png)
![Creare una chiave API](https://cdn-cn.cmdragon.cn/HQT5zxuf-P6VpJmrwRAfE.png)

### 2.4 Copiare la chiave API

- Copia la stringa della chiave che inizia con `sk-`
- **Importante**: Conserva questa chiave in modo sicuro, non potrai visualizzare nuovamente la chiave completa in seguito

---

## Passaggio 3: Configurare AI Magic Resume

Torna alla pagina di configurazione di AI Magic Resume e compila le seguenti informazioni:

| Campo          | Contenuto                           |
|-------------|--------------------------------|
| **Chiave API** | La chiave che hai copiato e che inizia con `sk-`              |
| **Nome fornitore**   | `openrouter`                   |
| **Endpoint API**  | `https://openrouter.ai/api/v1` |
| **ID modello**   | Seleziona un modello gratuito (vedi il prossimo passaggio)                   |

![Compilare le informazioni di configurazione](https://cdn-cn.cmdragon.cn/JRme33gZtIKB8x-zxZXw-.png)

---

## Passaggio 4: Selezionare un modello gratuito

### 4.1 Trovare modelli gratuiti

Sul sito web OpenRouter, inserisci **`free`** nella casella di ricerca dei modelli per filtrare tutti i modelli gratuiti.

![Cercare modelli gratuiti](https://cdn-cn.cmdragon.cn/EF-1D1wvVhMMWmtGRXD93.png)

![Copiare l'ID del modello](https://cdn-cn.cmdragon.cn/NKIeBOVFdhJUfXZ-SU7j3.png)

### 4.2 Copiare l'ID del modello

Seleziona qualsiasi modello gratuito e copia il suo ID completo (formato come: `google/gemma-3-27b-it:free`).

> ⚠️ **Nota**: I modelli gratuiti hanno limiti di utilizzo e non sono illimitati.

## Passaggio 6: Esportare e stampare il curriculum

### Esportare il curriculum in PDF

Fai clic sul pulsante **"Esporta curriculum"**, il sistema genererà automaticamente un file di curriculum in formato PDF standard.

![Esportare il curriculum](https://cdn-cn.cmdragon.cn/XtOsWjb5TvEadMHFbtgD5.png)

**Metodi di esportazione**:

- **Esportazione server**: Genera file PDF direttamente sul server, assicurando formato uniforme e layout bellissimo, pronto all'uso dopo il download
  ![Immagine](https://cdn-cn.cmdragon.cn/80GzYvsfodwhT4jzExPgs.png)
- **Stampa browser**: Chiama la funzione di stampa nativa del browser, può collegarsi alla stampante per stampare copie cartacee direttamente
  ![Immagine](https://cdn-cn.cmdragon.cn/v5gj00PLvRkg2Px9RKAEe.png)

**Scenari di utilizzo**:

- 📄 **Versione elettronica**: Per invio online, invio email, condivisione WeChat, ecc.
- 🖨️ **Versione cartacea**: Per colloqui, fiere del lavoro e altri scenari offline

**Suggerimento**: Si consiglia di anteprimere l'effetto PDF prima dell'invio, per assicurarsi che le informazioni chiave come informazioni di contatto ed esperienze di progetto siano complete e accurate.

## Domande frequenti

### Codice errore 429: Limite di frequenza delle richieste

Se incontri il seguente errore:

![Limite di frequenza](https://cdn-cn.cmdragon.cn/N8XAuxGLIdlIRaeAjkGMN.png)

```json
{
  "error": {
    "message": "Provider returned Error",
    "code": 429,
    "metadata": {
      "raw": "google/gemma-3-27b-it:free is temporarily rate-limited upstream. Please retry shortly, or add your own key to accumulate your rate limits: https://openrouter.ai/settings/integrations",
      "provider_name": "Google AI Studio",
      "is_byok": false
    }
  },
  "user_id": "user_2vDtkMeeWdLNDw24VQIEI0SunB3"
}
```

**Significato dell'errore**: Il modello gratuito utilizzato ha raggiunto il limite di frequenza di accesso.

**Soluzioni**:

1. **Attendere e riprovare**: Il limite per i modelli gratuiti è temporaneo, di solito attendi da alcuni minuti a ore o anche più a lungo.
2. **Cambiare modello**: Prova altri modelli gratuiti
3. **Usare modello a pagamento**: Rimuovi il suffisso `:free` dall'ID del modello e usa la versione a pagamento, richiede ricarica
4. **Vincolare la tua chiave API**: Vincola la tua chiave API di Google AI Studio nelle impostazioni di OpenRouter per usare una quota indipendente

## Suggerimenti per la sicurezza

- **La chiave API è un'informazione sensibile**, non condividerla con altri
- Si consiglia di salvare la chiave API in un luogo sicuro (come un gestore di password)
- Se sospetti una perdita di chiave, eliminala e ricreala immediatamente nel backend di OpenRouter
