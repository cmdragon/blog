## Шаг 2: Получение API-ключа OpenRouter

### 2.1 Открыть веб-сайт OpenRouter (на примере OpenRouter)

На странице настройки поставщика ИИ нажмите кнопку **"Получить API-ключ"**.

![Нажмите, чтобы получить API-ключ](https://cdn-cn.cmdragon.cn/syW9LMwLw0mBa1zn4p7DI.png)

Система откроет официальный веб-сайт OpenRouter в новой вкладке:

![Официальный веб-сайт OpenRouter](https://cdn-cn.cmdragon.cn/kJmBIJzNwEna_Dqixs5Gh.png)

### 2.2 Регистрация/Вход

- **Уже есть аккаунт**: Войдите напрямую
- **Нет аккаунта**: Нажмите зарегистрироваться

> 💡 **Совет**
> : Если вы не хотите использовать свою обычную электронную почту, вы можете воспользоваться нашим [сервисом временной электронной почты](https://tools.cmdragon.cn/zh/apps/temp-email) https://tools.cmdragon.cn/zh/apps/temp-email (убедитесь, что OpenRouter поддерживает временную электронную почту).

![Сервис временной электронной почты](https://cdn-cn.cmdragon.cn/Dn0WKPCb4GJRJzJTViZMl.png)
![Зарегистрировать аккаунт](https://cdn-cn.cmdragon.cn/zGVLipU7YiUpQ8LinfcsJ.png)

### 2.3 Создать API-ключ

После входа в систему следуйте этому пути:

1. Нажмите **"Settings"** в верхней панели навигации
2. Выберите **"API Keys"** в левом меню
3. Нажмите кнопку **"Create API Key"**

![Страница настроек](https://cdn-cn.cmdragon.cn/lKVZ6Umc3EkhQRowk7O_E.png)
![Управление API-ключами](https://cdn-cn.cmdragon.cn/B6lzexbzcIwpT6oltMdNr.png)
![Создать API-ключ](https://cdn-cn.cmdragon.cn/HQT5zxuf-P6VpJmrwRAfE.png)

### 2.4 Копировать API-ключ

- Скопируйте строку ключа, которая начинается с `sk-`
- **Важно**: Пожалуйста, сохраните этот ключ в безопасном месте, вы не сможете просмотреть полный ключ позже

## Шаг 3: Настройка AI Magic Resume

Вернитесь на страницу настройки AI Magic Resume и заполните следующую информацию:

| Поле          | Содержимое                           |
|-------------|--------------------------------|
| **API-ключ** | Ключ, который вы скопировали и начинается с `sk-`              |
| **Имя поставщика**   | `openrouter`                   |
| **Конечная точка API**  | `https://openrouter.ai/api/v1` |
| **ID модели**   | Выберите бесплатную модель (см. следующий шаг)                   |

![Заполнить информацию о настройке](https://cdn-cn.cmdragon.cn/JRme33gZtIKB8x-zxZXw-.png)

## Шаг 4: Выбор бесплатной модели

### 4.1 Найти бесплатные модели

На веб-сайте OpenRouter введите **`free`** в поле поиска моделей, чтобы отфильтровать все бесплатные модели.

![Поиск бесплатных моделей](https://cdn-cn.cmdragon.cn/EF-1D1wvVhMMWmtGRXD93.png)

![Копировать ID модели](https://cdn-cn.cmdragon.cn/NKIeBOVFdhJUfXZ-SU7j3.png)

### 4.2 Копировать ID модели

Выберите любую бесплатную модель и скопируйте ее полный ID (формат как: `google/gemma-3-27b-it:free`).

> ⚠️ **Примечание**: Бесплатные модели имеют ограничения по использованию и не являются безлимитными.

## Часто задаваемые вопросы

### Код ошибки 429: Ограничение частоты запросов

Если вы столкнулись со следующей ошибкой:

![Ограничение частоты](https://cdn-cn.cmdragon.cn/N8XAuxGLIdlIRaeAjkGMN.png)

```json
{
  "error": {
    "message": "Provider returned Error",
    "code": 429,
    "metadata": {
      "raw": "google/gemma-3-27b-it:free is temporarily rate-limited upstream. Please retry shortly, or add your own key to accumulate your rate limits: https://openrouter.ai/settings/integrations",
      "provider_name": "Google AI Studio",
      "is_byok": false
    }
  },
  "user_id": "user_2vDtkMeeWdLNDw24VQIEI0SunB3"
}
```

**Значение ошибки**: Используемая бесплатная модель достигла ограничения частоты доступа.

**Решения**:

1. **Подождать и повторить попытку**: Ограничение для бесплатных моделей является временным, обычно подождите от нескольких минут до часов или даже дольше.
2. **Сменить модель**: Попробуйте другие бесплатные модели
3. **Использовать платную модель**: Удалите суффикс `:free` из ID модели и используйте платную версию, требуется пополнение
4. **Привязать свой API-ключ**: Привяжите свой собственный API-ключ Google AI Studio в настройках OpenRouter для использования независимой квоты

## Советы по безопасности

- **API-ключ является конфиденциальной информацией**, пожалуйста, не делитесь им с другими
- Рекомендуется хранить API-ключ в безопасном месте (например, в менеджере паролей)
- Если вы подозреваете утечку ключа, пожалуйста, немедленно удалите его и создайте заново в бэкэнде OpenRouter
