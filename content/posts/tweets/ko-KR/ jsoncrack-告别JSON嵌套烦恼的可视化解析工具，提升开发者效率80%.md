이전에는 복잡한 JSON 데이터를 분석하기 위해 텍스트 편집기로 한 줄씩 계층 구조를 분석해야 했으나, 이는 저효율적이며 정보 과부하로 인한 오류 발생 위험이 높았습니다. **jsoncrack**은 이 문제를 해결하기 위해 추상적인 JSON 텍스트를 상호작용 가능한 그래픽 구조로 변환하여, 각 계층의 관계, 데이터 타입, 필드 논리를 시각적으로 표시해 계층 문제 탐색과 데이터 관계 정리 속도를 크게 개선합니다.

#### 2. JSON 데이터 가져오기 및 시각화 엔진 활성화

- **텍스트 붙여넣기**: 도구 내 입력창에 JSON 텍스트 직접 붙여넣기(다중 행 입력 및 구문 강조 지원)
- **온라인 주소 해석**: API 반환 주소와 같은 JSON 포함 URL 입력 → 자동으로 데이터 추출 및 해석

- **중앙 시각화 캔버스**: 노드 색상(문자열: 주황, 숫자: 파랑, 불리언: 초록, 중첩 객체: 회색)과 모양(값: 원, 객체: 사각형)으로 데이터 타입 및 계층 관계 직관적 구분. 마우스 드래그로 레이아웃 조정 가능
