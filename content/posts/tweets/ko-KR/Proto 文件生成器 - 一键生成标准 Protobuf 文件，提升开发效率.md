## ✨ 핵심 기능

### ✅ **시각적 인터페이스 설계, 학습 불필요**
- 직관적인 폼 기반 입력 인터페이스, 구문을 기억할 필요 없음
- 생성된 Proto 코드의 실시간 미리보기
- 드래그 앤 드롭으로 필드 순서 조정 가능
- 일반적인 필드 타입의 빠른 선택 내장

### ✅ **완전한 Protobuf 구문 지원**
- **메시지 정의**: 중첩 메시지, 반복 필드, 옵션 필드 지원
- **열거형 타입**: 열거값을 신속하게 정의, 번호 자동 할당
- **서비스 인터페이스**: RPC 메서드 정의, 스트리밍 호출 지원
- **가져오기 관리**: 가져오기 종속성을 자동 처리

### ✅ **Proto2/Proto3 이중 구문 지원**
- 원클릭으로 Proto2 와 Proto3 구문 전환
- 다른 버전의 구문 규칙에 자동 적응
- 버전 차이를 스마트하게 힌트
- 호환성 검사로 이전 문제 방지

### ✅ **지능형 코드 포맷팅**
- 자동 들여쓰기 및 정렬
- 통일된 코드 스타일
- 사용자 정의 포맷팅 규칙 지원
- 실시간 구문 오류 감지

### ✅ **배치 작업 및 내보내기**
- 여러 메시지 정의를 한 번에 생성
- .proto 파일 일괄 내보내기
- 파일 템플릿 저장 지원
- 생성 결과를 빠르게 복사
