## 2 žingsnis: Gauti OpenRouter API raktą

### 2.1 Atidaryti OpenRouter svetainę (naudojant OpenRouter kaip pavyzdį)

AI teikėjo konfigūravimo puslapyje spustelėkite mygtuką **"Gauti API raktą"**.

![Spustelėkite gauti API raktą](https://cdn-cn.cmdragon.cn/syW9LMwLw0mBa1zn4p7DI.png)

Sistema atidarys oficialią OpenRouter svetainę naujame skirtuke:

![OpenRouter oficiali svetainė](https://cdn-cn.cmdragon.cn/kJmBIJzNwEna_Dqixs5Gh.png)

### 2.2 Registruotis/Prisijungti

- **Jau turite paskyrą**: Prisijunkite tiesiogiai
- **Neturite paskyros**: Spustelėkite registruotis

> 💡 **Patarimas**
> : Jei nenorite naudoti savo įprasto el. pašto, galite naudoti mūsų [laikino el. pašto paslaugą](https://tools.cmdragon.cn/zh/apps/temp-email) https://tools.cmdragon.cn/zh/apps/temp-email (įsitikinkite, kad OpenRouter palaiko laikiną el. paštą).

![Laikino el. pašto paslauga](https://cdn-cn.cmdragon.cn/Dn0WKPCb4GJRJzJTViZMl.png)
![Registruoti paskyrą](https://cdn-cn.cmdragon.cn/zGVLipU7YiUpQ8LinfcsJ.png)

### 2.3 Sukurti API raktą

Prisijungę eikite šiuo keliu:

1. Viršutinėje navigacijos juostoje spustelėkite **"Settings"**
2. Kairiajame meniu pasirinkite **"API Keys"**
3. Spustelėkite mygtuką **"Create API Key"**

![Nustatymų puslapis](https://cdn-cn.cmdragon.cn/lKVZ6Umc3EkhQRowk7O_E.png)
![Valdyti API raktus](https://cdn-cn.cmdragon.cn/B6lzexbzcIwpT6oltMdNr.png)
![Sukurti API raktą](https://cdn-cn.cmdragon.cn/HQT5zxuf-P6VpJmrwRAfE.png)

### 2.4 Kopijuoti API raktą

- Nukopijuokite rakto eilutę, kuri prasideda `sk-`
- **Svarbu**: Išsaugokite šį raktą saugiai, vėliau negalėsite vėl peržiūrėti viso rakto

## 4 žingsnis: Pasirinkti nemokamą modelį

### 4.1 Rasti nemokamus modelius

OpenRouter svetainėje modelio paieškos laukelyje įveskite **`free`**, kad filtruotumėte visus nemokamus modelius.

![Ieškoti nemokamų modelių](https://cdn-cn.cmdragon.cn/EF-1D1wvVhMMWmtGRXD93.png)

![Kopijuoti modelio ID](https://cdn-cn.cmdragon.cn/NKIeBOVFdhJUfXZ-SU7j3.png)

### 4.2 Kopijuoti modelio ID

Pasirinkite bet kurį nemokamą modelį ir nukopijuokite visą jo ID (formatas kaip: `google/gemma-3-27b-it:free`).

> ⚠️ **Pastaba**: Nemokami modeliai turi naudojimo limitus ir nėra neribotai nemokami.

## 6 žingsnis: Gyvenimo aprašymo eksportavimas ir spausdinimas

### Eksportuoti PDF gyvenimo aprašymą

Spustelėkite mygtuką **"Eksportuoti gyvenimo aprašymą"**, sistema automatiškai sugeneruos standartinio PDF formato gyvenimo aprašymo failą.

![Eksportuoti gyvenimo aprašymą](https://cdn-cn.cmdragon.cn/XtOsWjb5TvEadMHFbtgD5.png)

**Eksportavimo būdai**:

- **Serverio eksportavimas**: Generuoti PDF failą tiesiai serveryje, užtikrinant vienodą formatą ir gražų išdėstymą, paruoštą naudoti po atsisiuntimo
  ![Paveikslėlis](https://cdn-cn.cmdragon.cn/80GzYvsfodwhT4jzExPgs.png)
- **Naršyklės spausdinimas**: Kvietia naršyklės vietinę spausdinimo funkciją, galima prijungti prie spausdintuvo ir tiesiogiai išvesti popierines kopijas
  ![Paveikslėlis](https://cdn-cn.cmdragon.cn/v5gj00PLvRkg2Px9RKAEe.png)

**Naudojimo scenarijai**:

- 📄 **Elektroninė versija**: Internetiniam pateikimui, siuntimui el. paštu, bendrinimui WeChat ir kt.
- 🖨️ **Popierinė versija**: Pokalbiams dėl darbo, darbo mugėms ir kitiems neprisijungus scenarijams

**Patarimas**: Prieš pateikiant rekomenduojama peržiūrėti PDF efektą, kad būtų užtikrinta, jog pagrindinė informacija, kaip kontaktinė informacija ir projektų patirtis, yra išsami ir tiksli.

## Dažniausiai užduodami klausimai

### Klaidos kodas 429: Užklausos dažnio riba

Jei susiduriate su šia klaida:

![Dažnio riba](https://cdn-cn.cmdragon.cn/N8XAuxGLIdlIRaeAjkGMN.png)

```json
{
  "error": {
    "message": "Provider returned Error",
    "code": 429,
    "metadata": {
      "raw": "google/gemma-3-27b-it:free is temporarily rate-limited upstream. Please retry shortly, or add your own key to accumulate your rate limits: https://openrouter.ai/settings/integrations",
      "provider_name": "Google AI Studio",
      "is_byok": false
    }
  },
  "user_id": "user_2vDtkMeeWdLNDw24VQIEI0SunB3"
}
```

**Klaidos reikšmė**: Naudojamas nemokamas modelis pasiekė prieigos dažnio ribą.

**Sprendimai**:

1. **Palaukti ir bandyti iš naujo**: Nemokamų modelių riba yra laikina, paprastai palaukite kelias minutes iki valandų ar net ilgiau.
2. **Pakeisti modelį**: Išbandyti kitus nemokamus modelius
3. **Naudoti mokamą modelį**: Pašalinti `:free` priesagą iš modelio ID ir naudoti mokamą versiją, reikia papildyti
4. **Susieti savo API raktą**: Susieti savo Google AI Studio API raktą OpenRouter nustatymuose, naudoti nepriklausomą kvotą
